DNA 백서(EN/KO) HTML 추가 패키지

대상 저장소 구조(사용자 스크린샷 기준):
/
  assets/
  en/
  ko/
  index.html
  sitemap.xml
  robots.txt

이 ZIP은 아래 파일/폴더를 추가(또는 교체)합니다:
- /index.html                         (루트 홈: Whitepapers 카드형 허브로 확장)
- /en/whitepapers/index.html          (EN 백서 목록 허브)
- /ko/whitepapers/index.html          (KO 백서 목록 허브)
- /en/whitepapers/dna-mm39/index.html (DNA 백서 EN: TOC sidebar 포함)
- /ko/whitepapers/dna-mm39/index.html (DNA 백서 KO: TOC sidebar 포함)
- /assets/whitepapers.css             (TOC/카드 레이아웃용 CSS)
- /assets/whitepapers.js              (모바일 TOC 열기/닫기)
- /SITEMAP_SNIPPET.xml                (기존 sitemap.xml에 추가할 URL 스니펫)

업로드 방법(GitHub Pages):
1) ZIP을 풀고, 저장소 루트에 폴더 병합 방식으로 업로드(기존 assets/style.css는 유지).
2) 루트 index.html은 교체됩니다. (원본 보관하려면 교체 전에 백업)
3) sitemap.xml에는 SITEMAP_SNIPPET.xml의 <url> 항목을 삽입하십시오.
