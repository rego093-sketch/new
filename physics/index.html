What this establishes

At the isostatic point (z = 2d = 6) the relaxed shear modulus is driven to zero along five independent observables while the bulk modulus stays finite, so a single longitudinal speed survives — c² = B/ρ. A rotational-unjamming inflow then fixes the proton charge radius as a stable attractor, and lattice counting fixes the proton-to-electron mass ratio, with no fitted parameter.

Scope & honesty

This is an alternative, reproducible interpretation — not an established replacement for the Standard Model. Results are graded Forced (geometry and integrals, no free coefficient), Calibrated (a single empirical length anchor, λ_ref = 632.99 nm, cross-checked to 0.16% by an independent geometric route), or Open (explicitly not claimed). The 3π⁴ → 6π⁵ ladder rests on one definitional identification; the fine-structure constant is an open input; and independent external re-execution of the code suite has not yet been performed.

Questions

Is this numerology?

No. Each relation is derived within a fixed lattice geometry from stated axioms and counting, with no adjustable coefficient, and is graded by how much is forced versus definitional. By an explicit rule, agreement with a measured value is never used to declare the underlying axioms true.

What does “the speed of light emerges” mean?

The packing self-organises to φ_jam ≈ 0.64 and passes through the isostatic point z = 2d = 6, where the relaxed shear modulus vanishes (five observables) and only the bulk modulus survives — leaving one longitudinal wave speed, c² = B/ρ.

How is it reproducible?

Every number is produced by deposited, versioned, SHA-256-checksummed code and data under a DOI, run through LOCK → Derive → Gate with explicit PASS / FAIL / INCONCLUSIVE verdicts.
